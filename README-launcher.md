# Team Office — лаунчер

## Два способа открыть офис

| Способ | Команда | Когда |
|--------|---------|-------|
| **Лаунчер (рекомендуется)** | `.\launch-office.bat` или `.\launch-office.ps1` | Обычная работа: одно окно, меньше RAM |
| **Браузер** | `.\start-office.ps1` → http://127.0.0.1:8765 | Запасной вариант, несколько вкладок, отладка |

Лаунчер и браузер — **один и тот же 3D-офис**. Лаунчер открывает Edge/Chrome в режиме `--app=` (без вкладок и панели).

## Лаунчер

```powershell
cd C:\Users\USER\office-hub
.\launch-office.ps1
```

- Поднимает сервер на порту **8765** (если ещё не запущен)
- Открывает окно с `?app=1&gfx=medium`
- В шапке появится метка **«приложение»**

### Варианты графики

```powershell
.\launch-office.ps1 -Gfx medium   # по умолчанию
.\launch-office.ps1 -Gfx low      # слабый ПК
.\launch-office.ps1 -Gfx high     # мощный GPU
.\launch-office.ps1 -Gfx eco2d    # 2D без WebGL
```

### Ярлык на рабочий стол

```powershell
.\install-game-shortcut.ps1
```

### Остановить сервер

```powershell
.\stop-office.ps1
```

## Новые возможности

### Чаты с папками

Слева в панели чата — список проектов. Кнопка **+** создаёт чат с названием и папкой (кнопка **Обзор…** или ручной путь).

Каждый чат — отдельная переписка. В Cursor уходит `folder_path` — агенты работают в выбранной папке.

Кнопка **Открыть в Cursor** — пробует `cursor`, `code`, затем Explorer.

### Read-only файлы (E1)

Агенты в Cursor могут читать папку чата через API: список файлов, `read` текста (до 512 KB), `git status`. См. skill `office-hub`.

### Ваш кабинет

В 3D-сцене — стол **«👤 Вы · Заказчик»** перед зоной собрания. Агенты подходят к вам, когда задают вопрос.

### Фото в чате

Кнопка **📎 Фото** — до **7 изображений** за одно сообщение (можно выбрать несколько сразу). Фото уходит в Cursor через мост `from-office.jsonl`.

### Вопросы от агентов

После **$team призвать** дизайнер может спросить (например: «Светлый / Средний / Тёмный»). Ответьте кнопкой или своим текстом.

Агенты в Cursor могут задать вопрос вручную:

```powershell
Invoke-RestMethod -Uri http://127.0.0.1:8765/api/cursor/inject-question -Method POST -ContentType "application/json" -Body '{"agent":"designer","text":"Какой стиль?","options":["Минимализм","Корпоративный"]}'
```

## LAN и телефон

Для доступа с телефона по Wi‑Fi используйте `.\start-office-lan.ps1` — лаунчер слушает только `127.0.0.1`.
